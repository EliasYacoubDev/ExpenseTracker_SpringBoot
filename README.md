# Expense Tracker API (Spring Boot)

A simple **Expense Tracking REST API** built using **Spring Boot 3**, **Java 17**, and **H2 in-memory database**.

This project demonstrates clean backend development using:

- REST API design
- Service → Repository → Entity architecture
- JPA + Hibernate (ORM)
- H2 in-memory DB for easy development
- Swagger UI API documentation

---

## Tech Stack

| Component     | Technology |
|---------------|------------|
| Backend       | Spring Boot 3.5.7 |
| Language      | Java 17 |
| Build Tool    | Maven |
| Database      | H2 (in-memory) |
| ORM           | Hibernate / Spring Data JPA |
| API Docs      | Swagger UI (springdoc-openapi) |
